# Plateforme E-Commerce avec Gestion de la Relation Client (CRM)

## Description

Ce projet consiste à développer une application web E-Commerce permettant aux utilisateurs de consulter des produits, gérer leur panier d'achat, passer des commandes et suivre leur état en ligne.

L'application intègre également des fonctionnalités de gestion de la relation client telles que le support client, la gestion des réclamations et le service après-vente (SAV).

Une interface d'administration permet de gérer les produits, les catégories, les utilisateurs, les commandes et les différentes demandes des clients.

## Fonctionnalités

### Espace Client

* Inscription et authentification
* Consultation des produits
* Gestion du panier d'achat
* Passage des commandes
* Suivi des commandes
* Gestion du profil utilisateur
* Dépôt de réclamations
* Demande de service après-vente (SAV)
* Contact avec le support client

### Espace Administrateur

* Gestion des utilisateurs
* Gestion des produits
* Gestion des catégories
* Gestion des commandes
* Gestion des réclamations
* Gestion du service après-vente
* Tableau de bord administrateur

## Architecture

L'application suit l'architecture MVT (Model - View - Template) de Django :

* **Model** : gère les données et la base de données.
* **View** : traite les requêtes et la logique métier.
* **Template** : affiche les interfaces utilisateur.
